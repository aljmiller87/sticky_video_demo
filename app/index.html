Documentation

Requirements

  1. jQuery: <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
  2. ScrollFire.js: jquery.scrollfire.js or jquery.scrollfire.min.js
  3. Vimeo player:
    npm: npm install @vimeo/player
    bower: bower install vimeo-player-js
    CDN: <script src="https://player.vimeo.com/api/player.js"></script>
  4. sticky video scss or css
  5. sticky video js

Calling

Syntax:

$('your-iframe').sticky_video(options);

Please note: ('your-iframe') must be an individual element. .sticky_video(options); can only call one element per html page.

Options (javascript object)

Width

Specify the dimension when video becomes fixed

width: 200px;

Location

Specify location via pixel count and direction: (top, right, bottom, left)

right: 30px,

bottom: 30px,

Please note that you cannot combine top with bottom or right and left at the same time.

Animation on Scroll

Specify if animation will take place on scroll

onScroll: true or false,

Default: default: true,

Animation on play

Specify whether animation will occur while video is playing

onPlay: true or false,

Default: default: true,

Animation on pause

Specify whether animation will occur while video is paused

onPause: true or false,

Default: default: false,
